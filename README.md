-Este proyecto consiste en el desarrollo de un sitio web utilizando HTML, cuyo objetivo es mostrar un explorador de proyectos con diferentes vistas. El sitio incluye varias páginas como inicio, listado de proyectos, detalle de cada proyecto y perfil de usuario.

-La estructura fue desarrollada utilizando etiquetas  como header, nav, main, section, article, aside y footer, permitiendo una organización clara del contenido. Se incorporaron formularios para el filtrado de proyectos, mejorando la interacción y la navegación dentro del sitio.
