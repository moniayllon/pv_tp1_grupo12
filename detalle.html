Descripcion

El objetivo principal de este proyecto es desarrollar una página web utilizando HTML y CSS que permita organizar y mostrar información de manera clara, estructurada y accesible para el usuario. Se busca aplicar correctamente etiquetas semánticas como header, main, section, article y footer, con el fin de construir una estructura lógica que facilite la comprensión del contenido y mejore la experiencia de navegación.

Además, otro de los objetivos es implementar estilos mediante CSS para lograr un diseño visual atractivo y funcional. Esto incluye la utilización de colores, tipografías y distribución de elementos que permitan una presentación ordenada de los proyectos. También se pretende comprender la importancia de la separación entre estructura y diseño, aplicando buenas prácticas en el desarrollo web y sentando bases sólidas para futuros proyectos más complejos.
